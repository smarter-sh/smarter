We've already obsessed over the details. Why should you?

The smarter.sh/v1 published JSON Schemas are a rock solid foundation for your OEM product. Leverage our JSON Schemas to turn your OEM product into your customers' Rock of Gibralter. Smarter takes care of the details for you. The only thing you have to do is to take advantage of our own attention to detail.

Check out our VS Code extension for Smarter manifests to get an idea of what you can create our published json schemas. This extension provides a rich editing experience for Smarter yaml manifest files, including syntax highlighting, autocompletion, context sensitive micro help, and validation against the JSON schema.

Validation

Leverage our JSON Schemas to validate your inbound requests against our data models. There's no better way to maximize your product's resiliency than by building on top of the JSON Schemas that we publish for all of our models.

Data Modeling

Our JSON Schemas act as a blueprint for your application's data models, ensuring consistency and correctness across different parts of your customers' systems.

Code Generation

Confidently and dynamically generate code for your data bindings, reducing the need to manually code data validations, and most importantly of all, reducing the potential for human error.

Documentation

Providing a clear specification of the JSON data format, which helps in understanding the structure and purpose of the data.

IDE Extension Development

Smarter JSON Schemas speed the development of IDE add-ons and extensions, like for VS Code for example. Use our JSON Schemas to create in-editor Smarter extensions for syntax auto-completion, tooltips, and error highlighting.

Updates
=======

Smarter is a Docker based application. The project maintains a Community Edition Docker
image on Docker Hub at `https://hub.docker.com/r/mcdaniel0073/smarter <https://hub.docker.com/r/mcdaniel0073/smarter>`_ that
is regularly updated with the latest security patches and feature enhancements.

Additionally, the Smarter project publishes a Helm Chart at `artifacthub.io - Project: Smarter <https://artifacthub.io/packages/helm/project-smarter/smarter>`_
that is regularly updated to support the latest Smarter Docker image versions.

Change logs are available on both repositories, as well as in the official Smarter GitHub repository:

- GitHub `CHANGELOG <https://github.com/smarter-sh/smarter/blob/main/CHANGELOG.md>`__
- GitHub `Releases <https://github.com/smarter-sh/smarter/releases>`_
- Docker Hub `Tags & Change Log <https://hub.docker.com/r/mcdaniel0073/smarter/tags>`_
- Artifact Hub `Changelog <https://artifacthub.io/packages/helm/project-smarter/smarter?modal=changelog>`__

It is recommended to regularly check for updates to ensure that your Smarter installation
remains secure and up-to-date with the latest features.

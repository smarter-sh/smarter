Trouble Shooting
================

Where to go for Help
----------------------

1. **Documentation**: Refer to the official Smarter documentation at `docs.smarter.sh <https://docs.smarter.sh/>`_ for guides, tutorials, and reference materials.
2. **Smarter Community Forum**: For general questions, discussions, and community support, visit the Smarter Community Forum at GitHub at `github.com/smarter-sh/smarter/discussions/ <https://github.com/smarter-sh/smarter/discussions/>`_.
3. **GitHub Issues**: If you encounter a bug or need to report an issue, please use the GitHub Issues page at `github.com/smarter-sh/smarter/issues/ <https://github.com/smarter-sh/smarter/issues/>`_.
4. **Contact The Author**: For direct support or inquiries, you can reach out to the project author, Lawrence McDaniel via email at `lpm0073@gmail.com <mailto:lpm0073@gmail.com>`_.
